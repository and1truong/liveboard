Embed LiveBoard on your own site2 files, ~20 lines total

Same-origin is required: shell and renderer postMessage validates against window.location.origin. Your index.html is the shell — drop it next to renderer.html and serve from any static host.

index.html (the shell) 
<!doctype html>
<html><head><meta charset="utf-8">
  <script type="module" crossorigin
    src="https://cdn.jsdelivr.net/npm/@wriven/liveboard-shell@0.20.1/dist-npm/assets/shell-DlF2Z-Xd.js"></script>
</head><body style="margin:0">
  <iframe id="renderer" style="width:100vw;height:100vh;border:0"></iframe>
  <script>
    window.__LIVEBOARD_CONFIG__ = { adapter: 'local', rendererUrl: './renderer.html' };
  </script>
</body></html>

</head><body><div id="root"></div></body></html>
