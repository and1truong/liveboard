liveboard · localStorage adapter
Embed LiveBoard on your own sitesingle file, ~20 lines

This page hosts the shell role inline — @wriven/liveboard-shared provides the broker and localStorage adapter, talking to the renderer iframe through postMessage. The iframe is just for CSS isolation; origin validation requires same-origin, so drop both files next to each other on any static host.

index.html (the shell) 
<!doctype html>
<html><head><meta charset="utf-8"></head>
<body style="margin:0">
  <iframe id="renderer"
    style="width:100vw;height:100vh;border:0"></iframe>
  <script type="module">
    import { Broker, LocalAdapter, BrowserStorage, shellTransport }
      from 'https://cdn.jsdelivr.net/npm/@wriven/liveboard-shared@0.20.1/dist/index.js';
    const iframe = document.getElementById('renderer');
    const transport = shellTransport(iframe, location.origin);
    new Broker(transport, new LocalAdapter(new BrowserStorage()),
      { shellVersion: '0.20.1' });
    iframe.src = './renderer.html'; // load renderer only after broker is ready
  </script>
</body></html>
