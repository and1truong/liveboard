## Start Here

- [ ] Read this card — you're looking at a Kanban board!
  priority: low
  Each column represents a stage. Cards move left to right as work progresses. This board has three columns: Start Here, Try It, and You Did It.

- [ ] Cards live inside columns
  priority: low
  Every card has a title (the text you see above) and can have a body (this text right here). Cards can also have tags, priorities, due dates, and assignees.

- [ ] Click any card to open it
  priority: low
  Clicking a card opens a detail view where you can edit the title, body, and all metadata. Try it on this card!

## Try It

- [ ] Drag a card from "Start Here" to this column
  priority: medium
  Grab any card by its handle and drag it here. This is how you move work between stages.

- [ ] Create a new card
  priority: medium
  Click the **+** button at the top of any column to add a new card. Give it a title and press Enter.

- [ ] Check off a completed task
  priority: medium
  Click the checkbox on any card to mark it as done. The card stays in place — you can move it to "You Did It" yourself.

- [x] This card is already done
  priority: low
